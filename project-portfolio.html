Personal Portfolio Website

This is the final version of my personal portfolio website project. It was created as part of a web development course using only HTML and CSS.

Project Overview

The goal of this project was to design a clean, responsive, and modern personal portfolio page that showcases my skills, background, and contact information. All animations and layouts were built using only CSS.

Key Features

  • Responsive layout for both mobile and desktop
  • Built entirely with HTML and CSS
  • Animated buttons and smooth transitions
  • Accessible design and clean typography
