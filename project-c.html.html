C Language Basics

This project showcases some of my early C programming exercises while learning the fundamentals of computer science. These programs helped me understand variables, conditionals, and basic arithmetic operations.

Example: Celsius to Fahrenheit Converter


#include <stdio.h>

int main() {
    float celsius, fahrenheit;

    printf("Enter temperature in Celsius: ");
    scanf("%f", &celsius);

    fahrenheit = (celsius * 9 / 5) + 32;

    printf("Temperature in Fahrenheit: %.2f\n", fahrenheit);
    return 0;
}

This simple program takes a temperature input in Celsius and converts it to Fahrenheit. It helped me understand how user input, arithmetic operations, and formatted output work in C.
